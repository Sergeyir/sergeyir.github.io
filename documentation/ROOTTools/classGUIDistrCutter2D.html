◆ WriteCutAreas()

template<typename T>
void GUIDistrCutter2D< T >::WriteCutAreas ( const std::string & outputFileName)

Write cut areas in the output file.

Writes in output file for every bin 0 or 1, i.e. whether the bin was or was not cut.

Parameters
[in]outputFileNamename of the output file in which the data will be written. If the file with the same name exists, the old file will be renamed to (outputFileName + ".tmp") and the data will be written in file (outputFileName).
